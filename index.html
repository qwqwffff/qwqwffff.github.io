🚀 راهنمای آپلود در گیت‌هاب و چت از راه دور

۱

ساخت مخزن (Repository) در گیت‌هاب

وارد حساب GitHub.com خود شوید. روی دکمه New Repository کلیک کنید. یک نام (مثلاً cyber-chat) انتخاب کرده و آن را Public کنید.

۲

آپلود فایل‌های همین پوشه

چون تمام پروژه در یک پوشه واحد قرار دارد، کافیست فایل‌های index.html، style.css و script.js را مستقیماً داخل مخزن آپلود (Upload files) کنید یا با دستورات زیر پوش کنید:

git init
git add .
git commit -m "Initial commit - Cyber Chat"
git branch -M main
git remote add origin https://github.com/YourUsername/cyber-chat.git
git push -u origin main
۳

فعال‌سازی رایگان سایت (GitHub Pages)

در صفحه مخزن گیت‌هاب خود، به بخش Settings > Pages بروید. در قسمت Branch گزینه main را انتخاب کرده و Save را بزنید. پس از ۱ تا ۲ دقیقه، سایت شما آنلاین و دارای آدرس اختصاصی خواهد شد!
